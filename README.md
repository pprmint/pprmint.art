# pprmint.art
The source code for my website. Shocker.

## Why?
Maybe looking at this code can help people getting started with their own website, or maybe they just wanna look out of curiosity. You tell me.

**This repository is not meant to make it easy for you to deploy copies of my website elsewhere as is.**

## How?
[Windows](https://www.microsoft.com/windows), [Visual Studio Code](https://code.visualstudio.com/), [pnpM](https://pnpm.io/), [React](https://reactjs.org/), [react-i18next](https://react.i18next.com/), [BabelEdit](https://www.codeandweb.com/babeledit), [MUI](https://mui.com/) and [coffee](https://www.amazon.de/dp/B001ILT920/).

(Mainly.)

## *How, but italic?*
`https://nodejs.org/` to download and install Node.js.

`npm install -g pnpm` to install pnpm.

`pnpm install` to download Node modules. ~~The warnings are a feature.~~

`pnpm start` to run a local dev server at [localhost:3000](localhost:3000).

`Go nuts` and play around with the code for yourself.
